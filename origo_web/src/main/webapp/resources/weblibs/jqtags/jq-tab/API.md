###Attributes

|Tag| Attribute | Value | Description |
|------|-----|------|---|
| jq-tab |   value  |  text    |  value of tab active |
| jq-tab-head|value|    text  | value to be set when selected  |
| jq-tab-pane|tab|    text  | value to be set when selected  |

|------|-----|-----|------|------|
